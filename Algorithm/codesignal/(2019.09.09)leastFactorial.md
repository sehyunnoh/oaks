> Eric's solution
>```js
>function leastFactorial(n) {
>  let result = 1, num = 0;
>  while (1) {
>    result *= ++num;
>    if(result >= n) break;
>  }
>  return result;
>}
>```

> Hyunju's solution
>```js
>function leastFactorial(n){
>  let i = 1, k = 1;
>  for(i; i <= n; i++){
>    k *= i;
>    if(k >= n) return k;
>  }
>}
>```
